AWS Machine Learning Workshop

A GUJCOST sponsored 6 days workshop was held on AWS Machine Learning in association with the Data Science Club, DSI, ACM and Gujarat Council of Science And Technology from 30 Jan’23 to 4 Feb’23.
The main highlights of the event were:
- ML Process, Tools and Challenges - Training model using Amazon Sagemaker 19 Hosting and Deploying with Deploying a model
- Processing time series data using AWS forecast is Image and Video Analysis with Amazon Rekognition
- Processing text for NLP
- Topic modeling, sentiment analysis and language
- Translation using AWS
- Earn an official AWS Badge and Certificate


Duration:

30 Jan - 4 Feb 2023


No. of Participants:

91 students
